## 北邮人团队copy页

这是一个面向招新的copy页，使用 Vue 3 + Vite + TailwindCSS 构建。我做了简单复刻与适配：

- 页面使用统一的浅灰背景，内容区域以白色卡片呈现；
- Hero 区支持拖拽视差，还可以通过左右按钮推动，交互更丝滑；
- “我们的产品”支持点击图标或左右箭头切换，选中项会显示圈线描边动画；
- 其他板块做了基础排版与响应式处理。

还有很多细节没完全打磨到位，但我愿意继续投入，把它做得更好看、更顺手。

### 开发

1. 安装依赖

```bash
npm install
```

2. 本地启动

```bash
npm run dev
```

3. 构建产物

```bash
npm run build
```
